 To-Do List Manager (Shell Script)
This is a simple To-Do List Manager built using Bash (Shell Script).  
It helps users to add, view, and manage daily tasks easily.

Features
- Add new tasks
- View all tasks
- Delete tasks
- Mark tasks as completed

Technologies Used
- Bash (Shell Script)
- Linux / Terminal
 ▶️ How to Run
1. Open terminal
2. Run the script:
   bash todo.sh
